# SERVER

Languages routes
 --

| HTTP VERB |             PATH                     |               DESCRIPTION                   |
|-----------|--------------------------------------|---------------------------------------------|
|GET        |'/api/languages/'                     | get all languages                           |
|POST       |'/api/languages/'                     | add new experience                          |
|DELETE     |'/api/languages/:experienceId'        | delete one experience                       |
